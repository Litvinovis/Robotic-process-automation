# 📢 Social Media Manager Script

Скрипт автоматизации публикаций в социальных сетях на базе [Clickermann](http://crapware.aidf.org/page/clickermann) v4.13

</div>

---

## О проекте

Скрипт берёт **10 новостей**, подготовленных по шаблону в Excel, и автоматически публикует их
сразу на нескольких платформах.

Реализован как автокликкер с привязкой к координатам экрана.
Для динамических элементов (лайки, подписки, добавление в друзья) применяется анализ экрана.

⏱️ Время выполнения: ~55 минут

---

## Платформы

| Платформа | Аккаунтов |
|-----------|-----------|
| ВКонтакте | 1 группа + 5 личных страниц |
| Twitter | 4 страницы |
| Facebook | 1 страница |
| Instagram | 1 страница |

---

## Требования

- [Clickermann](http://crapware.aidf.org/page/clickermann) версии **4.13**
- Microsoft Excel (для шаблона новостей)
- ОС Windows (Clickermann — Windows-only)

---

## Структура проекта

```
SMM manager/
├── Публикация новостей.cms  # Основной скрипт
├── SMMlibrary.cms           # Библиотека функций
├── NEWFOLLOW.bmp            # Шаблон для определения кнопки подписки
├── end.wav                  # Звук завершения
├── facebook.wav             # Уведомление: Facebook
├── instagram.wav            # Уведомление: Instagram
├── twitter.wav              # Уведомление: Twitter
└── VK.wav                   # Уведомление: ВКонтакте
```

---

## Использование

1. Заполнить Excel-шаблон (10 новостей)
2. Открыть `Публикация новостей.cms` в Clickermann
3. Запустить скрипт и не трогать мышь/клавиатуру ~55 минут
